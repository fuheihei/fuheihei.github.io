个人博客由hexo搭建

1.hexo项目构建

安装hexo，详细看hexo官网文档 https://hexo.io/zh-cn/docs/

环境：需要git，node环境。

hexo g #即 hexo generate，生成静态文件
hexo s #即 hexo server，本地运行服务器
hexo d #即 hexo deploy，部署服务器，常用指令 hexo clean && hexo d

2.主题选取

主题选择的是anatole

主题在hexo上的主题仓库: https://hexo.io/themes/

将主题复制到themes文件夹下，修改_config.yml中的属性

theme: anatole
# Extensions
## Plugins: https://hexo.io/plugins/
## Themes: https://hexo.io/themes/

3.写文章

hexo new [type] <文章名>
