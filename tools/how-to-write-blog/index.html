如何写博客

2022-04-16 tools 265 字
blog

我建立博客的初心是记录技术，并将自己在编程中学到的知识，写成规范的文档形式，尽力做到对自己对他人都是有帮助的。

1. 文档工具

文档文件格式还是markdown为主

2. markdown写作规范

写作规范详细参考github上的 Markdown 编写规范-中文

3. 字体

字体颜色，正常字体是默认的黑色，参考配色的颜色设计网站

1 - FlatUI 配色

2 - Ant Design 配色

红色 #fe6673 重要的内容

蓝色 #3498db 新名词首次出现

黄色 #f1c40f warning警告相关的颜色

绿色 #27ae60 论证过程，或者过程性质的代码

如下所示，在md中嵌入html标签

<span style="color:..."> </span> <!--用标签包裹需要上色的文字-->

4. 代码

代码部分格式以markdown样式为主，分为大的代码块（带行号）和普通代码行。

hello world //代码块

hello world 普通代码行

5. 总结

每天进步一点点，保持笔记的清晰有条理。
